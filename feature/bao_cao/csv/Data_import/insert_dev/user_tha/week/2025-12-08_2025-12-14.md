## 📐 Công thức tính và Xác minh

### HA Trung bình
```
SYS_TB = (125+138+126+122+140+128+124+136+125+120+...+125) / 19
       = 2448 / 19 = 129 mmHg

DIA_TB = (77+86+78+74+88+80+76+84+77+72+...+77) / 19
       = 1512 / 19 = 80 mmHg
```

### HA Cao nhất / Thấp nhất
```
HA Cao nhất = max(SYS) → 142 mmHg tại 11/12 13:00
            → Cặp: 142/90 mmHg
HA Thấp nhất = min(SYS) → 120 mmHg tại 11/12 04:00
            → Cặp: 120/72 mmHg
```

### % trong ngưỡng mục tiêu (SYS 120-130, DIA 70-80)
```
Trong ngưỡng = 13 lần (cả SYS và DIA đều trong ngưỡng)
Ngoài ngưỡng = 6 lần (#2: 138/86, #5: 140/88, #8: 136/84, #11: 142/90, #14: 135/83, ... (+1))

% = 13/19 × 100% = 68.4%
```

### ARV (Độ ổn định - Average Real Variability)
```
ARV = Σ|SYS[i+1] - SYS[i]| / (n-1)  [chỉ cặp trong 24h]

Tổng chênh lệch: 13+12+4+18+12+4+12+11+...+3 = 177
Số cặp hợp lệ: 17

ARV = 177 / 17 = 10.4 mmHg  [→ Biến động]
```

### ME diff (Morning-Evening difference)
```
Sáng (04:00-10:00): 125, 122, 124, 120, 123, ... (+1) → TB = 122.5 mmHg
Tối (20:00-24:00): 126, 128, 125, 127, 126, ... (+1) → TB = 126.2 mmHg

ME diff = 122.5 - 126.2 = -4 mmHg  [→ Cân bằng]
```

### Nhịp tim
```
HR_TB = (68+75+70+66+76+72+68+74+69+65+...+68) / 19
      = 1342 / 19 = 71 bpm

HR_max = 78 bpm
HR_min = 65 bpm
```
