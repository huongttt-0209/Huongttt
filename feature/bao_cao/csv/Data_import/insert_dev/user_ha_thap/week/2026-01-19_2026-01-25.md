DIA_TB = (61+62+60+61+60+63+59+61+58+62+...+62) / 16

```

            → Cặp: 96/63 mmHg
HA Thấp nhất = min(SYS) → 88 mmHg tại 22/01 04:30
            → Cặp: 88/58 mmHg
```

### % trong ngưỡng mục tiêu (SYS 120-140, DIA 70-90)
```
Trong ngưỡng = 0 lần (cả SYS và DIA đều trong ngưỡng)
Ngoài ngưỡng = 16 lần (#1: 94/61, #2: 95/62, #3: 92/60, #4: 94/61, #5: 93/60, ... (+11))

% = 0/16 × 100% = 0.0%
```

### ARV (Độ ổn định - Average Real Variability)
```
ARV = Σ|SYS[i+1] - SYS[i]| / (n-1)  [chỉ cặp trong 24h]

Tổng chênh lệch: 1+3+2+1+3+5+3+6+...+1 = 47
Số cặp hợp lệ: 15

ARV = 47 / 15 = 3.1 mmHg  [→ Ổn định]
```

### ME diff (Morning-Evening difference)
```
Sáng (04:00-10:00): 94, 94, 88, 96 → TB = 93.0 mmHg
Tối (20:00-24:00): 92, 96, 94, 92, 94 → TB = 93.6 mmHg

ME diff = 93.0 - 93.6 = -1 mmHg  [→ Cân bằng]
```
