## 📐 Công thức tính và Xác minh

### HA Trung bình
```
SYS_TB = (128+138+145+132+142+136+130+134+128+140) / 10
       = 1353 / 10 = 135 mmHg

DIA_TB = (82+89+92+85+91+88+84+86+82+90) / 10
       = 869 / 10 = 87 mmHg
```

### HA Cao nhất / Thấp nhất
```
HA Cao nhất = max(SYS) → 145 mmHg tại 15/12 21:00
            → Cặp: 145/92 mmHg
HA Thấp nhất = min(SYS) → 128 mmHg tại 15/12 04:30
            → Cặp: 128/82 mmHg
```

### % trong ngưỡng mục tiêu (SYS 120-140, DIA 70-90)
```
Trong ngưỡng = 8 lần (cả SYS và DIA đều trong ngưỡng)
Ngoài ngưỡng = 2 lần (#3: 145/92, #5: 142/91)

% = 8/10 × 100% = 80.0%
```

### ARV (Độ ổn định - Average Real Variability)
```
ARV = Σ|SYS[i+1] - SYS[i]| / (n-1)  [chỉ cặp trong 24h]

Tổng chênh lệch: 10+7+13+10+6+6+4+12 = 68
Số cặp hợp lệ: 8

ARV = 68 / 8 = 8.5 mmHg  [→ Ổn định]
```

### ME diff (Morning-Evening difference)
```
Sáng (04:00-10:00): 128, 130 → TB = 129.0 mmHg
Tối (20:00-24:00): 145, 142, 128 → TB = 138.3 mmHg

ME diff = 129.0 - 138.3 = -9 mmHg  [→ Cân bằng]
```

### Nhịp tim
```
HR_TB = (71+77+80+73+78+75+72+74+71+77) / 10
      = 748 / 10 = 75 bpm

HR_max = 80 bpm
HR_min = 71 bpm
```
