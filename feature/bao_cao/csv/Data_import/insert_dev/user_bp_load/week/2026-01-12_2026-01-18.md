## 📐 Công thức tính và Xác minh

### HA Trung bình
```
SYS_TB = (138+142+138+145+135+148+130+140+136+134+...+134) / 16
       = 2196 / 16 = 137 mmHg

DIA_TB = (89+91+89+92+87+95+84+90+88+86+...+86) / 16
       = 1412 / 16 = 88 mmHg
```

### HA Cao nhất / Thấp nhất
```
HA Cao nhất = max(SYS) → 148 mmHg tại 13/01 21:00
            → Cặp: 148/95 mmHg
HA Thấp nhất = min(SYS) → 128 mmHg tại 17/01 04:30
            → Cặp: 128/82 mmHg
```

### % trong ngưỡng mục tiêu (SYS 120-140, DIA 70-90)
```
Trong ngưỡng = 12 lần (cả SYS và DIA đều trong ngưỡng)
Ngoài ngưỡng = 4 lần (#2: 142/91, #4: 145/92, #6: 148/95, #11: 142/91)

% = 12/16 × 100% = 75.0%
```

### ARV (Độ ổn định - Average Real Variability)
```
ARV = Σ|SYS[i+1] - SYS[i]| / (n-1)  [chỉ cặp trong 24h]

Tổng chênh lệch: 4+4+7+10+13+18+10+4+...+2 = 102
Số cặp hợp lệ: 14

ARV = 102 / 14 = 7.3 mmHg  [→ Ổn định]
```

### ME diff (Morning-Evening difference)
```
Sáng (04:00-10:00): 138, 145, 130, 128 → TB = 135.2 mmHg
Tối (20:00-24:00): 138, 148, 136, 142, 132, ... (+1) → TB = 138.3 mmHg

ME diff = 135.2 - 138.3 = -3 mmHg  [→ Cân bằng]
```

### Nhịp tim
```
HR_TB = (77+78+77+80+75+82+72+77+76+74+...+74) / 16
      = 1217 / 16 = 76 bpm

HR_max = 82 bpm
```
